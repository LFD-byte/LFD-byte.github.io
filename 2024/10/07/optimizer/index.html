Optimizer

本文主要介绍了深度学习模型中主要使用的几个 optimizer (优化器)。

SGD

Algorithm Stochastic gradient descent (SGD) update at training iteration k

Require: Learning rate \(\epsilon_k\)

Require: Intial parameter \(\pmb{\theta}\) while stopping criterion not met do Sample a minibatch of \(m\) examples from the training set \(\{x^{(1)},\dots,x^{(m)} \}\) with corresponding targets \(y^{(i)}\) Compute gradient estimate: \(\hat{\pmb{g}} \leftarrow +\frac{1}{m} \nabla_{\pmb{\theta}}\sum_i L(f(x^{(i)};\pmb{\theta}),y^{(i)})\) Apply update: \(\pmb{\theta} \leftarrow \pmb{\theta} - \epsilon \hat{\pmb{g}}\) end while

Adagrad

Algorithm The AdaGrad algorithm

Require: Global learning rate \(\epsilon\)

Require: Initial parameter \(\pmb{\theta}\)

Require: Small constant \(\delta\), perhaps \(10^{-7}\), for numerical stability Initialize gradient accumulation variable \(\pmb{r}=0\) while stopping criterion not met do Sample a minibatch of \(m\) examples from the training set \(\{x^{(1)},\dots,x^{(m)} \}\) with corresponding targets \(y^{(i)}\) Compute gradient estimate: \(\hat{\pmb{g}} \leftarrow \frac{1}{m} \nabla_{\pmb{\theta}}\sum_i L(f(x^{(i)};\pmb{\theta}),y^{(i)})\) Accumulate squared gradient: \(\pmb{r} \leftarrow \pmb{r} + \pmb{g} \odot \pmb{g}\) Compute update: \(\Delta \pmb{\theta} \leftarrow -\frac{\epsilon}{\delta+\sqrt{\pmb{r}}} \odot \pmb{g}\). (Division and square root applied element-wise) Apply update: \(\pmb{\theta} \leftarrow \pmb{\theta} + \Delta \pmb{\theta}\) end while

RMSProp

Algorithm The RMSProp algorithm

Require: Global learning rate \(\epsilon\), decay rate \(\rho\)

Require: Initial parameter \(\pmb{\theta}\)

Require: Small constant \(\delta\), usually \(10^{-6}\), used to stabilize division by small numbers Initialize gradient accumulation variable \(\pmb{r}=0\) while stopping criterion not met do Sample a minibatch of \(m\) examples from the training set \(\{x^{(1)},\dots,x^{(m)} \}\) with corresponding targets \(y^{(i)}\) Compute gradient estimate: \(\hat{\pmb{g}} \leftarrow \frac{1}{m} \nabla_{\pmb{\theta}}\sum_i L(f(x^{(i)};\pmb{\theta}),y^{(i)})\) Accumulate squared gradient: \(\pmb{r} \leftarrow \rho \pmb{r} + (1- \rho)\pmb{g} \odot \pmb{g}\) Compute update: \(\Delta \pmb{\theta} = -\frac{\epsilon}{\sqrt{\delta+\pmb{r}}} \odot \pmb{g}\). (\(\frac{1}{\sqrt{\delta+\pmb{r}}}\) applied element-wise) Apply update: \(\pmb{\theta} \leftarrow \pmb{\theta} + \Delta \pmb{\theta}\) end while

Adam

Algorithm The Adam algorithm

Require: Step size \(\epsilon\) (Suggested default: 0.001)

Require: Expontential decay rates for moment estimates, \(\rho_1\) and \(\rho_2\) in \([0,1)\). (Suggested defaults: 0.9 and 0.999 respectively)

Require: Small constant \(\delta\) used for numerical stabilization. (Suggested default: \(10^{-8}\))

Require: Initial parameters \(\pmb{\theta}\) Initialize 1st and 2nd moment variables \(\pmb{s}=0,\pmb{r}=0\) Initialize time step \(t=0\) while stopping criterion not met do Sample a minibatch of \(m\) examples from the training set \(\{x^{(1)},\dots,x^{(m)} \}\) with corresponding targets \(y^{(i)}\) Compute gradient estimate: \(\hat{\pmb{g}} \leftarrow \frac{1}{m} \nabla_{\pmb{\theta}}\sum_i L(f(x^{(i)};\pmb{\theta}),y^{(i)})\) \(t\leftarrow t+1\) Update biased first moment estimate: \(\pmb{s} \leftarrow \rho_1 \pmb{s} + (1 - \rho_1)\pmb{g}\) Update biased second moment estimate: \(\pmb{r} \leftarrow \rho_2 \pmb{r} + (1-\rho_2)\pmb{g} \odot \pmb{g}\) Correct bias in first moment: \(\hat{\pmb{s}} \leftarrow \frac{\pmb{s}}{1-\rho_1^t}\) Correct bias in first moment: \(\hat{\pmb{r}} \leftarrow \frac{\pmb{r}}{1-\rho_2^t}\) Compute update: \(\Delta \pmb{\theta} = -\epsilon \frac{\hat{\pmb{s}}}{\sqrt{\hat{\pmb{r}}+\delta}}\). (\(\frac{1}{\sqrt{\delta+\pmb{r}}}\) applied element-wise) Apply update: \(\pmb{\theta} \leftarrow \pmb{\theta} + \Delta \pmb{\theta}\) end while
